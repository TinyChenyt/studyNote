# 路由配置

## 安装react路由库
运行`npm install react-router-dom`安装react路由库

安装成功如图所示：
![20250630115016](https://tiny-blog.oss-cn-guangzhou.aliyuncs.com/blog/20250630115016.png)

## 创建路由文件
在src目录下，新建一个router文件夹，在router文件夹下，新建一个index.js文件，用于配置路由。

### 创建基础路由组件Home 和 Setting
![20250630115739](https://tiny-blog.oss-cn-guangzhou.aliyuncs.com/blog/20250630115739.png)

## 在router中引入配置路由组件
```js
import { createBrowserRouter } from "react-router-dom";

// pages
import Home from "../pages/Home";
import Setting from "../pages/Setting";

const router = createBrowserRouter([
    {
        path: "/",
        element: <Home />
    },
    {
        path: "/setting",
        element: <Setting />
    }
]);

export default router;
```
