# 常用命令

## 切换目录
```shell
# 切换到指定目录
cd /Users/yang/Documents/GitHub/

# 切换到上一级目录
cd ../

# 切换到根目录
cd /
```

## 查看当前目录
```shell
pwd
```

## 查看当前目录下所有文件
```shell
# 显示所有文件
ls
# 显示隐藏文件
ls -l
# 显示文件大小
ls -a
# 显示文件类型
ls -la
# 显示文件时间
ls -h
# 显示文件排序
ls -s
# 显示文件倒序
ls -t
```

## 清理屏幕
```shell
clear
```

## 创建目录
```shell
mkdir test
```

## 删除目录
```shell
rm -rf test
```

## 创建文件
```shell
touch test.txt
```

## 删除文件
```shell
rm test.txt
```
