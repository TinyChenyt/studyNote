# 项目初始化

**版本说明：**

- node：18.14.2

## 创建react项目
使用命令 `npx create-react-app project-name` 创建项目

创建成功打开项目，项目目录如下所示
![20250630113206](https://tiny-blog.oss-cn-guangzhou.aliyuncs.com/blog/20250630113206.png)

创建的是react+js项目，可根据需求选择react+ts或者react+js

运行`npm start`运行项目

**根据实际需要下载对应依赖**
