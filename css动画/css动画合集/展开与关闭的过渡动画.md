# 展开与关闭的过渡动画

## 使用display实现
优点：简单，兼容性良好
缺点：没有动画效果，展示与隐藏比较突兀

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>

    <link rel="stylesheet" href="./index.css">
</head>
<body>
    <div class="container">
        <ul class="name">
            <p class="name_act">name1</p>
            <li class="name_item act">name1</li>
            <li class="name_item">name2</li>
            <li class="name_item">name3</li>
            <li class="name_item">name4</li>
            <li class="name_item">name5</li>
            <li class="name_item">name6</li>
        </ul>
    </div>
</body>
</html>
```
```css
* {
    margin: 0;
    padding: 0;
}

li {
    list-style: none;
}

.container {
    margin: 20px 0;
    display: flex;
    align-items: center;
    justify-content: center;
}

.name {
    width: 120px;
    height: 30px;
    transition: all 0.5s;
    cursor: pointer;
}

.name_act {
    border: 1px solid rgba(0, 0, 0, 0.5);
    height: 30px;
    line-height: 30px;
    text-align: center;
}
.name_item {
    height: 30px;
    text-align: center;
    border: 1px solid rgba(0, 0, 0, 0.5);
    border-top: 1px solid transparent;
    font-size: 20px;
    color: #000;
    cursor: pointer;
    display: none;
}
.name:hover .name_item {
    display: block;
}
.name_item.act {
    color: #13abd1;
    border: 1px solid #17e63a;
    border-top: 1px solid transparent;
    background: rgba(23, 230, 58, 0.1);
}
.name_item:hover {
    color: #13abd1;
    border: 1px solid #17e63a;
     border-top: 1px solid transparent;
    background: rgba(23, 230, 58, 0.1);
}
```
